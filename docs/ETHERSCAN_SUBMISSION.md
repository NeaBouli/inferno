6. **Project Description:**

   Inferno ($IFR) is a deflationary ERC-20 utility token on Ethereum Mainnet (deployed March 5, 2026).

   **MECHANICS:**
   - 2.5% permanent burn on every transfer (2% sender burn + 0.5% recipient burn)
   - 1% pool fee routed to protocol contracts (FeeRouterV1)
   - No mint function — supply can ONLY decrease
   - Current supply: ~998.5M (from 1B genesis)
   - ~1.5M IFR burned since launch

   **UTILITY — LOCK-TO-ACCESS:**
   - Users lock IFR in IFRLock contract
   - Locked IFR = lifetime premium access to integrated builder products
   - No subscription, no recurring payment
   - One-time lock = permanent access

   **FAIR LAUNCH:**
   - No presale, no VC, no private sale
   - Community Bootstrap active until June 5, 2026
   - Anyone can contribute 0.01–2 ETH
   - Bootstrap funds Uniswap V2 liquidity pool

   **GOVERNANCE:**
   - 48-hour timelock on all parameter changes
   - Treasury: Gnosis Safe 3-of-5 multisig
   - LP Reserve: 400.6M IFR in Gnosis Safe 3-of-5
   - All contracts open source + verified on Etherscan

   **INTEGRATED BUILDERS:**
   - K-9 Academy (dog training platform)
   - StealthX/SecureCall (encrypted voice calls)
   - Vendetta (price transparency network)
   - NEXUS GR (B2B/P2P marketplace — coming soon)
