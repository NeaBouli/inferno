Phased LP Strategy

400M IFR werden maximalen Wert erschaffen

1. Warum NICHT sofort alle 400M in den Pool?

Die Uniswap V2 AMM-Formel ist einfach: x * y = k. Jeder Trade verändert das Verhältnis der beiden Reserven, und der Preis ergibt sich aus y / x.

Beim Bootstrap-Preis P0 = $0.0000002 sind 400M IFR gerade einmal $80 wert.

Entscheidend: LP hinzufügen ändert den Preis nicht. Man muss das bestehende Verhältnis beibehalten. Aber 400M IFR für $80 einzusetzen ist eine katastrophale Verschwendung.

Kernproblem: Pool = 100M IFR + 0.01 ETH. Adding 400M IFR + 0.04 ETH = 500M IFR + 0.05 ETH. Preis unverändert. Aber 0.04 ETH verschwendet für einen 5x tieferen Pool bei nur $20 Tiefe. 

// AMM Constant Product Formula
x * y = k

// Bei P0 = $0.0000002
// 400,000,000 IFR * $0.0000002 = $80

// Pool vorher:  100M IFR + 0.01 ETH   (k = 1,000,000)
// LP hinzu:    +400M IFR + 0.04 ETH   (gleiches Ratio)
// Pool nachher: 500M IFR + 0.05 ETH   (k = 25,000,000)
// Preis:        0.05 / 500M = $0.0000001 ETH/IFR  ← UNVERÄNDERT
// Verschwendet: 0.04 ETH für 5x Tiefe bei ~$20 Gesamttiefe

2. Die 4 Phasen

PHASE 0 — Nach Bootstrap

Initial Pool: 100M IFR + ETH (~$20 Tiefe)

  • Team fügt KEIN LP hinzu
  • Schutz durch CommitmentVault + LendingVault
  • Organisches Wachstum durch Community-Trades
MILESTONE 1 — Pool = 1 ETH (~$2,000)

Governance Proposal: +50M IFR + 0.5 ETH

  • Pool danach: ~150M IFR + 1.5 ETH ($3,000 Tiefe)
  • 99% kaufen kostet: ~$30,000 statt $2,000
  • Erheblich manipulations-resistenter
MILESTONE 2 — Pool = 5 ETH (~$10,000)

Governance Proposal: +100M IFR + proportionales ETH

  • Pool deutlich tiefer
  • Manipulation wird ökonomisch unrentabel
  • Professionelle Market-Making-Tiefe
MILESTONE 3 — September 2026

LiquidityReserve Unlock (200M IFR)

  • Governance entscheidet über Menge
  • Preis ist jetzt deutlich höher als P0
  • Gleiche 200M IFR = massiv mehr Liquidität als bei P0

3. Effizienz-Beweis

Die gleichen 400M IFR erzeugen bei höherem Preis exponentiell mehr Liquidität:

Szenario 400M IFR Wert Pool-Tiefe
Bei P0 ($0.0000002) $80 $0.016
Bei P0 × 1,000 ($0.0002) $80,000 $1,600,000

GLEICHE 400M IFR = 100,000,000x mehr Liquidität. Warten ist die einzig rationale Strategie.

4. LP Reserve Safe

Alle 400.6M IFR der Liquidity Reserve liegen in einem Gnosis Safe mit 3-of-5 Multisig:

  • 3 Signaturen nötig für jede LP-Zugabe
  • Öffentliche Ankündigung vor jeder LP-Zugabe
  • Kein einzelnes Team-Mitglied kann LP allein bewegen
