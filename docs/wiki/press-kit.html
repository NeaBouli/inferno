Press Kit & Media Resources

Key facts, media assets, and brand resources for journalists and content creators.

Inferno ($IFR) is a deflationary ERC-20 utility token on Ethereum. Every transfer permanently burns tokens, reducing supply over time. Launched via Community Fair Launch Model (CFLM) — no presale, no VC funding, no private allocation. Users lock IFR on-chain to unlock lifetime access to partner products and services.

Deflation Mechanism

Every IFR transfer applies the following fees, enforced at the smart contract level:

Fee Rate Action
Sender Burn 2.0% Permanently burned from sender
Recipient Burn 0.5% Permanently burned from received amount
Pool Fee 1.0% Routed to FeeRouterV1 (BurnReserve / BuybackVault)
Total 3.5% Hard cap: 5% maximum (on-chain enforced)

Lock-to-Access Model

Users lock IFR tokens on-chain via the IFRLock contract to gain access tiers across all partner products:

Tier Lock Amount Access Level
Bronze 1,000 IFR Basic premium features
Silver 5,000 IFR Extended premium features
Gold 25,000 IFR Full premium access
Platinum 100,000 IFR Lifetime all-access

Builder Ecosystem

  • PartnerVault — 40M IFR reserved for builder rewards, distributed when users lock IFR through partner integrations.
  • Lock-triggered Creator Rewards — when a user locks IFR via a partner, the partner receives a percentage from the pool.
  • Per-partner cap prevents pool exhaustion by any single partner.
  • Annual emission cap (4M IFR/year) ensures long-term sustainability.
  • Builder tokens grant future DAO voting rights (Phase 4+).

Governance

  • 48-hour Timelock on all parameter changes — no instant admin actions.
  • Guardian role can cancel malicious proposals but cannot execute them.
  • No token voting (prevents whale manipulation and flash-loan attacks).
  • Multisig transition (Phase 3) followed by full DAO governance (Phase 4).
  • All governance actions are transparent and verifiable on Etherscan.
