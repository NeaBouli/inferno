Inferno ($IFR) Documentation

Complete technical reference for the Inferno token ecosystem — contracts, tokenomics, governance, security, and integration guides.

Welcome

This wiki is the single source of truth for the Inferno ($IFR) project. It covers every aspect of the protocol: from the smart contract architecture and fee mechanics to governance proposals, security audits, and step-by-step integration guides for developers.

Inferno is a Community Fair Launch Model (CFLM) ERC-20 token on Ethereum with a built-in deflationary fee structure, on-chain governance via a 48-hour timelock, and a transparent vesting schedule. Whether you are a holder, developer, or auditor, you will find the information you need here.

Inferno Protocol is fully open source and community-owned. No single entity controls the protocol. All contracts are verified on Ethereum Mainnet, governed by a 48-hour timelock, and protected by Gnosis Safe multisig. The protocol lives on-chain — permanently.
