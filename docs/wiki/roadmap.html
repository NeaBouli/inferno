Roadmap

Six-phase development plan from foundation to full DAO governance.

Current status: Phase 4 (Mainnet Launch) — 9 contracts deployed and verified on Ethereum Mainnet (March 2026). Token distribution complete (1B IFR, CFLM). Next milestones: LP pairing, LP lock, ownership transfer to Gnosis Safe multisig.

Phase 1: Foundation Done

Q4 2025 — Core protocol contracts and testing infrastructure.

  • InfernoToken (ERC-20, 9 decimals, fee-on-transfer)
  • LiquidityReserve (6-month lock, 50M/quarter)
  • Vesting (12-month cliff, 36-month linear)
  • BuybackVault (50/50 split, 60-day activation)
  • BurnReserve (permanent burn + tracking)
  • Hardhat test suite (core contracts)
  • Slither security audit (0 high/critical)

Phase 2: Governance & Lock Done

Q1 2026 — On-chain governance, utility lock, Sepolia deployment.

  • Governance contract (48h timelock, propose/execute/cancel)
  • IFRLock contract (lock/unlock, isLocked(), multi-app lockType)
  • Sepolia deployment (14 contracts, all verified)
  • Uniswap V2 LP pairing (IFR/WETH)
  • Ownership transfer to Governance
  • Governance Proposals #0–#3, #7–#9 executed
  • Token Dashboard (React 18 + Vite)
  • Governance Dashboard (React 18 + TypeScript + Tailwind)
  • 367 contract tests, 444 total tests

Phase 3: Ecosystem In Progress

Q1–Q2 2026 — Partner ecosystem, applications, audit preparation.

  • PartnerVault v2 (authorizedCaller, anti-double-count, algo throttle)
  • FeeRouterV1 (EIP-712 vouchers, protocol fee routing)
  • Benefits Network (QR-based lock verification)
  • Creator Gateway (YouTube × IFR Lock Bridge)
  • Points Backend (SIWE auth, EIP-712 voucher issuance)
  • AI Copilot (RAG knowledge base, 3 modes)
  • Partner Directory (filter by type/status)
  • Lock UI in Token Dashboard
  • 14 wiki pages, transparency report
  • Ownership Transfer (v2 redeploy + Proposals #7–#9 executed)
  • LP Token Lock/Burn
  • Professional security audit

Phase 4: Mainnet Launch In Progress

Q1–Q2 2026 — Production deployment and partner onboarding.

  • Gnosis Safe Multisig deployed (1-of-1, Phase 0)
  • Mainnet deploy (9 contracts, all verified on Etherscan)
  • Token distribution complete (1B IFR, CFLM)
  • Fee exemptions configured (8 addresses)
  • Mainnet LP pairing (IFR/ETH on Uniswap V2)
  • LP Token lock/burn
  • Ownership transfer to Governance (mainnet)
  • Dashboards switched to mainnet
  • First partner integrations
  • CoinGecko / CoinMarketCap listing

Phase 5: Growth & DAO Planned

Q3 2026 – Q1 2027 — Partner ecosystem expansion and DAO transition.

  • Partner onboarding (Tier 1–3 applications)
  • Multisig upgrade (2-of-3 → 3-of-5 → 4-of-7)
  • Guardian multisig (2-of-3, cancel-only)
  • Creator Rewards live (automated)
  • FeeRouter adapter ecosystem
  • DAO governance framework (token-weighted voting)
  • Partner token voting rights
  • Bug bounty program (Immunefi)

Phase 6: Maturity Planned

Q2 2027+ — Full decentralization and ecosystem maturity.

  • Full DAO transition (Admin → Multisig → DAO)
  • Cross-protocol integrations
  • Advanced analytics dashboard
  • Buyback vault activation
  • Community grants program
  • CEX integration support (fee-exempt listings)

Current Focus

Phase 4 completion — mainnet contracts deployed and verified. Next steps: LP pairing, LP lock, ownership transfer to Gnosis Safe multisig. See Mainnet Checklist for detailed progress.

All timelines are estimates. Security audit completion is a hard prerequisite for mainnet — launch will not proceed without it.
