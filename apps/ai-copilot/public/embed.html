IFR Copilot Embed Example

Add this snippet to any HTML page to embed the IFR Copilot widget:

<!-- IFR Copilot Widget -->
<script>
  (function() {
    var iframe = document.createElement('iframe');
    iframe.src = 'https://ifr-copilot.yourdomain.com';
    iframe.style.cssText = 'position:fixed;bottom:20px;right:20px;width:400px;height:600px;border:none;z-index:9999;border-radius:16px;box-shadow:0 8px 32px rgba(0,0,0,0.4)';
    document.body.appendChild(iframe);
  })();
</script>

Replace ifr-copilot.yourdomain.com with your deployed URL.
